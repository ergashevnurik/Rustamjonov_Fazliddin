ABOUT US

HTML

HTML (from the English HyperText Markup Language - "Hypertext Markup Language") is a standardized document markup language for viewing web pages in web pages. Web browsers receive an HTML document from the server via the HTTP/HTTPS protocol or disable the context disk, then interpret the code in the interface that will be displayed on the monitor.

CSS

Cascading Stylesheets — or CSS — is the first technology you should start learning after HTML. While HTML is used to define the structure and semantics of your content, CSS is used to style it and lay it out. For example, you can use CSS to alter the font, color, size, and spacing of your content, split it into multiple columns, or add animations and other decorative features.

JAVA SCRIPT

Examples of programs created in JavaScript are programs that check user-entered data and perform certain actions when opening or closing a document. These programs follow user-generated instructions, such as pressing the mouse button, entering information on the screen, or moving the mouse across the page. Programs in JavaScript can also control the size of the browser and the attributes of the document.
